# LIBERTE-SE: O Guia Definitivo Para Largar o Cigarro

Uma landing page para um ebook que ajuda as pessoas a parar de fumar, abordando a dependência física, psicológica e comportamental do tabagismo.

## Sobre o Projeto

Este projeto é uma landing page para promover um ebook chamado "LIBERTE-SE", destinado a ajudar fumantes a abandonarem o cigarro de forma definitiva. A página apresenta:

- Benefícios de parar de fumar
- Como funciona o método apresentado no ebook
- Conteúdo do ebook (10 capítulos)
- Depoimentos e resultados baseados em ciência
- Seção FAQ
- Formulário para compra

## Tecnologias Utilizadas

- HTML5
- CSS3
- Design responsivo

## Como Visualizar

Abra o arquivo `main.html` em qualquer navegador moderno para visualizar a landing page.
